#  Calculator 

This is a simple Calculator Web Application developed as part of my Web Development Internship (Task 3).

##  Features
- Perform basic arithmetic operations:
  - Addition (+)
  - Subtraction (−)
  - Multiplication (*)
  - Division (/)
- Clear (C) button to reset input
- Delete (←) button to remove last value
- User-friendly interface
- Responsive layout using CSS Grid

##  Technologies Used
- HTML
- CSS
- JavaScript

##  Project Structure
- calculator.html
- style.css
- calculator.js

##  How to Use
1. Open `calculator.html` in your browser
2. Click buttons to enter values
3. Press `=` to get result
4. Use `C` to clear and `←` to delete

##  Learning Outcome
- Improved understanding of JavaScript functions
- Learned DOM manipulation and event handling
- Practiced CSS Grid for layout design
